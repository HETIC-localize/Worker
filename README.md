# HETIC-localize/Worker

### Requirements

- go ^1.15
- mysql ^8.0

### Tools

- [GCP-PubSub](https://cloud.google.com/pubsub/docs/create-topic-client-libraries)
- [DeepL](https://www.deepl.com/docs-api)

- Config `.env` file like `.env.sample`
- Run `go get`

- Run `go run .`
